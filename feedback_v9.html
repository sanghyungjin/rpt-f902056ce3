1. 연구 개요

1.1 v9 의 설계 동기

v8 종료 시점에 외부 DSP 전문가로부터 받은 3가지 피드백에 대응하여 v9 를 설계했습니다:

  1. Provenance discipline 부족 — 이전 시리즈 (v7/v8) 의 일부 상위 후보가 합성 cross-breed 산물로 자연/합성 구분이 어려움 [2]14
  2. Human DSP single source 강조 — 비교종은 conservation 계산용 contrast group 일 뿐이며, 실제 추출 source 는 Human DSP 단독이어야 함 [4]16
  3. 자연 3-10mer 의 IP / 합성 / provenance 면 재평가 필요 — 14-mer 합성에 가려진 짧은 자연 motif 의 별도 평가 가치

v9 의 대응: 자연 sliding window only (mutation/cross-breed 일체 없음), Human P15924 only (3-10mer 고정), alignment-based conservation map 재구축 [1][2].

1.2 중요 Disclaimer — DSP 와 DEJ 의 관계

Desmoplakin 은 desmosome 의 plaque protein (cell-cell 접착) [1][2] 이며, basement membrane zone (BMZ) / dermal-epidermal junction 의 직접 anchoring 분자가 아닙니다. DEJ 의 직접 구조 단백질은 collagen IV (lamina densa), collagen VII (anchoring fibril), laminin-332, nidogen-1/2, integrin α6β4, BP180 (COL17A1), BP230 등입니다.

본 연구의 "DEJ 관련" 표현은 간접 지지 (basal keratinocyte cytoskeleton 안정화 → 기저막 접착 유지) 를 의미합니다. 피부 노화 맥락의 DEJ 평탄화 [15] 는 핵심 표적이며, DEJ_V8 의 "DEJ triad" 명명은 간접 기여 (Type I collagen 합성 [10], MMP 억제 [3], desmosomal/keratin network 지지) 를 의미합니다.

Wet-lab 설계 시 주의: COL17A1 / laminin-332 직접 결합 assay 는 적합하지 않으며, (a) Type I procollagen ELISA [8], (b) basal keratinocyte adhesion/migration (HaCaT/NHEK), (c) MMP-1/3/9 inhibition zymography [3][8] 를 권장합니다.

3. 피드백 요청 항목

3.1 자연 추출 전략에 대한 평가

v9 는 v1-v8 의 합성 mutation/cross-breed 접근에서 자연 sliding window only 로 전환했습니다. 전문가 의견을 요청합니다:

  • 자연 short peptide (3-10mer) 의 cosmetic 응용 가치는 합성 14-mer 대비 어떻게 평가하십니까? [11]12
  • v9 의 median DEJ_V8 0.4943 vs v8 의 median 0.6176 (-0.12 gap, Mann-Whitney U=0, p≈0) — 이 score gap 은 자연 제약의 합법적 penalty 인가 아니면 wet-lab 무의미 수준인가?
  • v9 와 v8 의 substring containment: v9 Top 100 의 23% 가 v8 14-mer 의 substring (예: HPR ⊂ KGHKHRGSHPRINT) — pair experiment 로 길이별 활성 비교를 해볼 만한 가치가 있나요?

3.2 Conservation Map 재구축 타당성

v1-v8 까지 사용된 기존 v4 conservation map 이 naive position-wise zip() 계산으로 max=0.5, mean=0.067 의 비현실적 분포를 낸 버그 상태였습니다 [4][16]. v9 에서 BioPython PairwiseAligner + BLOSUM62 로 재구축:

  • Alignment-based conservation (self-alignment gate + domain sanity gate 통과) 의 접근 타당성?
  • Podarcis lilfordi 제외 결정 — 3가지 signal (length 76.6%, InterPro feature count 1, v4-phase0 central rod conservation 0.388) 기반 [16]. 외부 검증 필요 항목이 있나요?
  • Human + 2 lizard (Anolis + P. muralis) 만으로 DSP-like conservation 을 충분히 측정할 수 있나요? 추가 비교종이 필요하다면 어떤 종?

3.3 DEJ_V8 의 RK-fraction bias 해석

v9 pool 6,366 peptide 분석 결과 Pearson r (RK-fraction ↔ DEJ_V8) = +0.866 (극도로 높음). Component 별:

  • cpp r=+0.93, wound_healing_sim r=+0.73, dsp_interaction r=+0.59 (DEJ-ER 포함)
  • DEJ-ER triad r=+0.598 (DEJ_V8 raw 대비 절반 수준)

질문:

  • FusPBESM2 의 CPP head (r=+0.93) 는 의도된 charge bias (CPP 정의상) — 허용 가능 인가 완화 필요 인가? 6
  • wound_healing_sim head r=+0.73 은 예상 외 — LL-37 [9] / Tβ4 [11] 등 cationic-rich training data 편향 가능성? WH head 재가중 또는 재학습 필요?
  • v10 에서 RK hard cap 도입 시 어느 level 이 적절? (현재 cationic 비율 0.40 부근에서 saturation)

3.4 3-mer wet-lab 권고 조건부 보류

v9 의 Top 1 GSR, Top 5 HPR, Top 13 HRL 은 3-mer. 초안에서 wet-lab 1차 후보로 철회한 이유:

  1. 자유 3-mer 자연 피부 반감기 최악 — aminopeptidase / DPP4 / kallikrein / cathepsin 에 의해 빠르게 분해 7
  2. FusPBESM2 3-mer pool median 0.19 (vs 5-10mer median 0.30-0.34) — OOD 가능성
  3. Top 5 HPR 은 이미 v8 synthetic KGHKHRGSHPRINT 의 substring — 새 발견 아님

질문:

  • 3-mer 자유 peptide 의 stabilization strategy (N-acetyl, D-amino acid, cyclization, carrier system) 중 현실적으로 적용 가능한 것은? 12
  • 3-mer 단독 합성 대신 5-10mer context window 재탐색 권고 (GSR → C-term SSAPGSRSGSRSGSRSGSRSGSRSGSRRGSFDAT @2820-2853 영역, HPR → N-term MSCNGGSHPRINT 영역) — 타당한가?
  • FusPBESM2 의 3-mer calibration 검증 방법 제안?

3.5 IP / FTO landscape 의 반직관적 finding

v9 pool 의 27% 가 pataa BLAST 에서 full-hit BLOCKED [5]. 자연 추출이 novelty-safe 할 것이라는 직관과 반대입니다.

  • DSP 같은 잘 연구된 단백질의 자연 short subsequence 가 1/4 이상 prior art — 이 finding 의 타당성 평가?
  • Type I blind spot 이슈: v9 의 FTO BLAST 는 length ≥ 6 만 — 3-5mer 은 not_applicable_short_peptide skip. Figure 5 참조 — pool 46%, Top 100 24% 가 blind spot. 이를 어떻게 보완?
  • cosmetic 산업에서 DSP-derived peptide 의 실제 특허 landscape 에 대한 전문가 의견?

3.6 Wet-lab assay design 우선순위

v9 가 제안한 assay priority (DSP ≠ DEJ disclaimer 반영):

  1. HIGH — HaCaT/NHEK Type I procollagen ELISA [8]10
  2. HIGH — MMP-1/3/9 zymography 3
  3. MEDIUM — HaCaT scratch assay (wound healing) [7]11
  4. MEDIUM — Calcein-AM uptake (CPP / membrane permeability) 6
  5. LOW — DPPH/ABTS antioxidant
  6. LOW — Franz cell skin permeation

질문:

  • 위 priority 가 v9 의 Top 100 (PRD 59 / Rod 38 / plakin 3) 에 적합한가요?
  • COL17A1/laminin-332 직접 결합 assay 는 DSP 가 직접 BMZ anchor 가 아니므로 negative control 로만 권장 — 이 판단이 타당한가?
  • 추가로 필요한 in vitro assay (예: keratin IF network visualization, desmosome assembly marker) 가 있나요?

3.7 v10 Synthetic Mutation Phase 설계 방향

v9 가 자연 baseline 을 확립했으므로 v10 은 mutation 을 통한 score ceiling 돌파:

  1. v9 CLEAR + non-rod + RK-frac < 0.3 의 ~20-25 개 → BLOSUM62 single/double substitution
  2. Cross-breed 시 strict provenance (match_type="chimera" + 두 parent exact position)
  3. RK hard cap + MC Dropout wire
  4. 5-10mer 범위 우선 (3-4mer OOD 회피)

  • 2-3mer 영역의 예측 신뢰도 는 model 재학습 (short peptide 전용) 없이는 개선 불가한가?
