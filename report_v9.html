1. 연구 배경 및 목적

1.1 타겟 단백질: Desmoplakin (DSP)

Figure 1 — v9 alignment-based conservation map 의 per-residue distribution. 첨부 파일 fig1_conservation_histogram.png 참조.

Desmoplakin (DSP) 은 desmosome — 표피 각질세포 (keratinocyte) 간 mechanical junction — 의 중심 plaque protein 으로, keratin intermediate filament (IF) 를 세포막 cadherin complex 에 anchoring 하여 조직의 기계적 무결성을 담당합니다 [1]. DSP 기능 이상은 심근병증 (AD cardiomyopathy), 탈모, 표피 수포증 등 다양한 임상 phenotype 과 연관되며, 조건부 표피 DSP 결손 (conditional epidermal Dsp deletion) 실험에서 epidermal sheet 가 mechanical stress 하에서 fragmentation 을 보여 DSP 가 keratinocyte 간 mechanical integrity 에 필수적임이 확인되었습니다 [2].

Human DSP (UniProt P15924) 는 2,871 amino acids 로 구성된 대형 단백질이며, 다음 3 개 주요 도메인으로 구분됩니다 [14]:

도메인 위치 (aa) 구조 기능
N-terminal plakin 1-1056 Spectrin repeats + SH3 Plakoglobin/Plakophilin 결합 → desmosomal cadherin tail 과 연결
Central rod 1057-1756 Parallel dimeric coiled-coil DSP homodimer 형성, rod 강성 제공
C-terminal PRD 1757-2871 Plakin Repeat Domain A/B/C Keratin IF 결합 site (PRD-B 가 core binding 영역)
Note: 본 연구의 모든 펩타이드는 Human DSP P15924 단일 source 로부터 natural sliding window 로 추출되며 (mutation / cross-breed 없음), 비교종은 conservation 계산을 위한 contrast group 으로만 사용됩니다.

1.2 Dermal-Epidermal Junction (DEJ) 상세 구조

피부 DEJ 는 표피 basal keratinocyte 와 진피 papillary dermis 를 연결하는 basement membrane zone (BMZ) 으로, 다음 4 개 구역으로 구성됩니다:

구역 핵심 분자 역할
Hemidesmosome Integrin α6β4, BP180 (COL17A1), BP230, Plectin Basal keratinocyte 를 BMZ 에 고정
Lamina lucida Laminin-332, Nidogen-1/2 Hemidesmosome ↔ lamina densa 연결
Lamina densa Type IV collagen, Perlecan BMZ 의 주요 구조 단백질 층
Sub-lamina densa Type VII collagen (anchoring fibril) BMZ 를 진피 Type I/III collagen ECM 에 tethering

DEJ flattening (rete ridges 의 소실) 은 photoaged skin 의 특징적 형태학적 변화로, 진피-표피 계면 강도 저하 및 주름 발생의 구조적 원인입니다 [15].

Note: 본 연구의 "collagen activity" score 는 Type I collagen (진피 ECM 의 약 80% 를 차지) 합성 촉진 proxy 입니다. DEJ 의 직접 구조를 이루는 Type IV/VII 와는 구별되며, 자세한 DSP–DEJ 관계 disclaimer 는 §1.5 를 참조하십시오.

1.3 Desmosome Assembly Pathway

DSP 가 참여하는 desmosome 조립은 Ca²⁺ 의존적 다단계 과정입니다 [1]:

단계 분자 역할
1 Desmoglein 1/3 + Desmocollin 1/3 Transmembrane cadherin cluster 형성 (homophilic + heterophilic trans binding)
2 Plakoglobin + Plakophilin-1/3 Cadherin tail 에 결합, intracellular plaque core 형성
3 Desmoplakin (DSP) N-terminal plakin domain 으로 plaque 에 anchor, C-terminal PRD 가 keratin IF 에 결합
4 Keratin IF (K5/K14 basal, K1/K10 suprabasal) Cell-cell junction 을 cytoskeletal network 에 통합

DSP 의 이중 결합 (plaque side N-term + IF side C-term PRD) 은 desmosome 의 mechanical continuity 에 필수적이며, DSP 유래 펩타이드는 원론적으로 이 인터페이스의 일부를 mimic 또는 stabilize 할 가능성이 있습니다. 단, 짧은 3-10mer 가 거대 단백질의 구조적 역할을 온전히 재현하는지는 실험적 검증을 요구합니다 (§1.9 참조).

1.4 피부 노화의 분자 메커니즘

피부 노화는 intrinsic (chronological) 과 photoaging 의 두 축으로 진행되며, 공통 핵심 cascade 는 다음과 같습니다 [3] [8]:

  1. ROS 축: UV / 환경 스트레스 → mitochondrial ROS 및 H₂O₂ 증가 → AP-1 전사인자 활성화 → MMP-1/3/9 발현 증가 → Type I/III collagen 분해 → 진피 ECM 손실.
  2. Inflammaging 축: 만성 저강도 염증 → NF-κB 활성화 → TNF-α, IL-6, IL-1β 분비 → MMP 추가 유도 + keratinocyte 분화 이상 → DEJ 구조 약화.
  3. TGF-β 억제: Aging 피부에서 TGF-β signaling 감소 → fibroblast 의 pro-collagen 합성 저하 → ECM 재생 부전.
  4. MMP imbalance: MMP vs TIMP 비율 증가 → lamina densa Type IV collagen 및 papillary dermis Type VII anchoring fibril 저해 → DEJ flattening [15].
전략적 대응: DEJ_V8 scoring 은 이 cascade 의 각 축에 가중치를 할당합니다 — Collagen 0.22 (Type I 합성), MMP Inhibition 0.12 (protease 억제), Anti-Inflammatory 0.12 (NF-κB 축), Antioxidant 0.08 (ROS 축), Wound Healing similarity 0.10, DSP interaction 0.18. DEJ-ER triad meta score (0.22·col + 0.18·dsp + 0.12·mmp) / 0.52 는 이 중 DEJ 회복과 가장 직접적으로 연관된 3 개 차원만으로 wet-lab 우선순위를 재계산합니다.

1.5 DSP 와 DEJ 의 관계 — 중요 disclaimer

Desmoplakin 은 desmosome plaque protein (cell-cell 접착) 이며, DEJ / BMZ 의 직접 anchoring 분자가 아닙니다.

§1.2 에서 기술한 바와 같이 DEJ 의 직접 구성 분자는 Type IV collagen (lamina densa), Type VII collagen (anchoring fibril), laminin-332, nidogen-1/2, integrin α6β4, BP180 (COL17A1), BP230 입니다 [1]. Desmoplakin 은 이 복합체에 포함되지 않습니다.

본 연구의 "DEJ 관련 회복" 표현은 다음의 간접 지지 (indirect support) 경로를 의미합니다:

1. Basal keratinocyte 내부 keratin IF–desmosome 네트워크 안정화 → 세포가 hemidesmosome 을 통해 BMZ 에 부착하는 mechanical foundation 제공.
2. DSP 유래 펩타이드가 collagen 합성 (Type I, ECM 재건), MMP 억제 (BMZ 분해 방지), 항염 (NF-κB 축 차단) 활성을 보이는 경우 — 이는 DEJ 의 주변 환경을 개선하는 효과.

Wet-lab 설계 시 COL17A1 / laminin-332 직접 결합 assay 가 아닌 (a) Type I collagen 합성 측정, (b) basal keratinocyte adhesion / migration assay, (c) MMP-1/3/9 inhibition assay 가 적합합니다.

1.6 Cross-species Conservation 전략

Figure 2 — N-terminal plakin / Central rod / C-terminal PRD 도메인별 mean conservation 비교. 첨부 파일 fig2_domain_conservation.png 참조.

가설: 인간-도마뱀 간 진화적으로 보존된 DSP 도메인에서 추출한 펩타이드는 수억 년의 functional constraint 하에 유지되었으므로, 생물학적 활성 예측의 신뢰도가 높다.

이 가설은 진화적 보존이 functional constraint 의 proxy 라는 일반 원칙 (comparative genomics) 에 기반합니다. Anolis carolinensis 의 꼬리 재생 transcriptome 분석은 vertebrate developmental + wound repair pathway (Wnt, FGF, MAPK 등) 의 광범위한 활성화를 보고했으며 [4], 재생 조직의 mechanical reassembly 단계에서 keratinocyte junction 의 재형성이 일반적으로 동반됨을 시사합니다 (재생 능력 자체는 DSP 단일 단백질 기능보다 광범위한 분자 네트워크의 결과). v9 의 lizard 비교종 선택은 DSP 가 재생을 driving 한다는 강한 인과 주장이 아니라, 억 단위 시간 척도에서 DSP 서열이 보존된 위치는 functional constraint 가 강한 영역일 가능성이 높다는 보존-활성 가설의 근거로 활용됩니다.

v9 는 alignment-based conservation map 을 위해 Human + 2 lizard species 를 사용합니다:

단백질 UniProt 종 길이 역할
Human DSP P15924 Homo sapiens 2,871 aa Reference (추출 source)
Anolis DSP H9G6P9 Anolis carolinensis 2,879 aa Query 1 (Iguania, ~150 Mya 분기)
P. muralis DSP A0A670I745 Podarcis muralis 2,797 aa Query 2 (Lacertoidea)

Conservation map 은 BioPython PairwiseAligner (global mode, BLOSUM62, open_gap=−10, extend_gap=−0.5) 기반으로 재구축되었으며, self-alignment (Human vs Human = 1.0) 및 domain-level sanity gate (N-plakin 0.77, Rod 0.72, C-PRD 0.78) 를 모두 통과했습니다. 상세한 재구축 근거는 §2.1 을 참조하십시오.

1.7 연구 목표

본 연구의 목표는 다음 네 가지입니다:

  1. Human DSP 자연 3-10mer 의 체계적 재평가 — 길이 3 에서 10 까지의 모든 sliding window subsequence 를 DEJ_V8 / DEJ-ER 로 scoring 하여, 이전 시리즈에서 14-mer 합성 후보에 가려져 있던 짧은 자연 motif 의 활성 잠재력을 분리 평가합니다.
  2. Alignment-based conservation map 구축 — 이전에 사용되던 naive zip() 기반 position-wise 비교를 BLOSUM62-scored global alignment 로 대체하여, frame shift artifact 없이 per-residue conservation 을 재계산합니다.
  3. DEJ-ER triad meta score 도입 — 9-component DEJ_V8 가 RK-rich 서열에 과도하게 반응하는 bias (Pearson r(RK-fraction, DEJ_V8) = 0.866) 를 완화하기 위해, 회복과 직접 연관된 collagen + dsp + mmp 3 개 차원만의 meta score 를 wet-lab ranking 에 사용합니다.
  4. Freedom-to-Operate 및 provenance 검증된 wet-lab short list 확보 — 각 후보에 대해 domain origin, conservation score, pataa BLAST, internal cosmetic/bioactive DB hit 정보를 동시에 제공하여, 합성 전 IP 리스크와 mechanism 해석의 불확실성을 최소화합니다.

2. 연구 방법론

2.1 Conservation Map 재구축 (v1_alignment_based)

v1-v8 까지 사용된 기존 v4 conservation map 의 결함:

  • 계산 방식: 각 종 서열을 Human 과 position-wise zip() 으로 비교 → frame shift 발생
  • 결과: max=0.5, mean=0.067 의 비현실적 분포
  • 영향: v3-v8 의 conservation feature 가 사실상 noise

v9 의 재구축:

  • BioPython Align.PairwiseAligner (global mode, BLOSUM62 substitution matrix)
  • open_gap_score=-10, extend_gap_score=-0.5, terminal gaps=0
  • Self-alignment gate (Human vs Human = 1.0 bit-perfect) 통과
  • Domain sanity gate: N-plakin 0.7661, Central rod 0.7236, C-PRD 0.7794 (모두 통과)

2.2 Natural Sliding Window Extraction

for k in range(3, 11):           # k-mer length 3-10
    for i in range(len(DSP) - k + 1):
        peptide = DSPi:i+k
        if avg_cons[i:i+k] >= 0.9 AND min_cons[i:i+k] >= 0.5:
            candidates.append(peptide)

Dual conservation threshold (avg ≥ 0.9 AND min ≥ 0.5) — single threshold 로는 "평균은 높지만 특정 position 이 완전 unconserved" 인 peptide 가 통과하는 false-positive 문제 방지.

Figure 3 — Dual conservation threshold (avg, min) sensitivity sweep. avg≥0.9 + min≥0.5 가 best operating point — 더 엄격하면 pool 너무 작고, 느슨하면 false-positive 급증. 첨부 파일 fig3_threshold_sweep.png 참조.

Provenance discipline: 모든 peptide 가 P15924 의 정확한 sub-sequence 임을 보장. 이전 시리즈에서 발생했던 합성 cross-breed/chimera 가 v9 pool 에 진입할 경로가 구조적으로 차단.

2.4 FTO 분석 (Freedom-to-Operate)

  • Reference peptide database cross-check: 6 source bidirectional substring matching against established cosmetic + bioactive peptide databases (agingbase, topicalpdb, biopep_uwm, satpdb, dbaasp, apd3 — 약 50,000+ entries 합산)
  • pataa BLAST: PAM30, word_size=2 (length ≥ 6 만, 3-5mer 는 short-peptide skip)
  • Classification: BLOCKED / CAUTION / CLEAR + pcpc_status (REGISTERED / PATENT_FULL_HIT / PURE_NOVEL 등)

3.5 핵심 발견 — 자연 추출이 자동으로 IP-clean 하지 않음

v9 pool 의 27% 가 BLOCKED [1]. 즉 DSP 같은 잘 연구된 단백질의 자연 short subsequence 의 1/4 이상은 이미 타 기업/연구소가 광범위 특허화. "자연 추출 = novelty-safe" 라는 직관은 틀렸습니다.

이는 후속 모든 natural extraction 연구의 workflow 에서 FTO 분석을 extraction 과 병렬 실행. (Katayama et al. 1993 의 procollagen pentapeptide [10] 같은 단순 motif 도 patent 화 되어 있는 사례 다수) 해야 함을 시사. 사후에 FTO 를 맞추려 하면 top candidates 의 1/4 이 떨어져 나갑니다.
