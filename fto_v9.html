2.3 핵심 발견 — "자연 추출 = novelty-safe" 가정 반박

v9 pool 의 **27% (1,721/6,366)** 가 BLOCKED. 즉 DSP 같은 잘 연구된 단백질에서 추출한 자연 short peptide 의 1/4 이상은 이미 타 기업·연구소가 광범위 특허화했습니다. **"자연 서열은 자동으로 IP-clean" 이라는 직관적 가정은 틀렸습니다.**

이 finding 의 함의:

  1. **Natural ≠ novel** — 잘 연구된 단백질은 short subsequence 도 patent landscape 가 dense. 4-6mer 짧은 motif 는 진화적 보존 영역이 화장품/cosmeceutical 특허의 표적이 되어 있을 가능성이 높습니다.
  2. **Cross-version 비교** — v8 의 14-mer synthetic mutation pool 은 novel mutation 으로 patent hit 이 적은 반면, **natural short peptide pool 은 prior art density 가 높습니다**. 즉 합성 cross-breed 가 IP risk 측면에서 일부 우위를 갖습니다.
  3. **Workflow 함의** — 자연 추출 연구는 FTO 분석을 **extraction 과 병렬 실행** 해야 합니다. 사후에 FTO 를 맞추려 하면 top candidates 의 1/4 이 떨어져 나갑니다.

5. Type I FTO Blind Spot — pataa BLAST 단축 펩타이드 한계

**Figure 5** — FTO Type I Blind Spot: pataa BLAST short-peptide skip zone. Pool 의 46% (2,929/6,366) 와 Top 100 의 24% (24/100) 가 length 3-5 이므로 BLAST 가 적용되지 않습니다.

NCBI BLASTp 의 `pataa` 데이터베이스는 **length ≥ 6** 의 query 만 의미 있는 결과를 반환합니다 (PAM30 + word_size=2 setting 에서 length 5 미만은 background noise). 따라서 v9 pool 의 length 3-5 peptides 는 다음 처리가 적용됩니다:

Length 처리 한계
3-5 `not_applicable_short_peptide` (pataa BLAST skip) 외부 특허 검사 불가 — 내부 DB 검사로만 분류
6-10 정상 BLAST 적용 특허 100% match → BLOCKED, partial → CLEAR

**함의**: Length 3-5 peptide 의 CLEAR 분류는 "내부 cosmetic / bioactive DB 에 없음" 만 보장하며, "외부 특허 등록 없음" 은 보장하지 않습니다. 짧은 peptide 의 wet-lab 합성 전에는 별도 patent search (USPTO, EPO, KIPRIS) 가 필수입니다.
